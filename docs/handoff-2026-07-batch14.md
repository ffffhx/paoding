## N1：菜谱分享图卡（传播场景，竞品少有）

1. 菜谱详情页「生成分享图」：前端 Canvas 绘制竖版长图（9:16 或 3:4）——菜名大标题、封面图（有步骤图取第一张，无图用主题色块）、食材清单（两栏）、步骤摘要（编号短句，超长截断）、底部落款「庖丁 · 把做菜视频拆成看得懂的为什么」+ 分享页链接文字。
2. 导出 PNG 下载（`canvas.toBlob`）；移动端走系统分享（`navigator.share` 可用时带图分享，不可用回退下载）。
3. 布局计算（文本换行/分栏/截断/画布高度）抽纯函数配测试；Canvas 绘制本身不强测。
4. 暗色无关（图卡用固定浅色主题，保证可读）。i18n 文案。

## N2：做饭统计页（Parsnip 式轻激励）

1. 「我的厨房」统计区（可挂在设置页或独立入口）：本周/本月做饭次数（来自做过打卡的时间戳）、累计做过菜数、最常做的 3 道菜、收藏数、掌握技法数（做过的菜命中的技法去重计数，复用技法库数据）、连续做饭天数（streak）。
2. 全部从 userdata/菜谱本地计算，纯函数（输入打卡记录 → 统计对象）配测试；不加新端点。
3. 打卡数据若缺时间戳（老数据），计入累计但不计入周/月/streak，如实显示。
4. 轻量卡片样式，暗色适配，i18n。

## N3：桌面键盘快捷键（桌面响应式补全）

1. 全局：`/` 聚焦搜索、`Esc` 关弹窗（已有的保持）；跟做模式：`←/→` 或 `j/k` 上下步、`空格` 暂停/继续当前计时器、`r` 朗读当前步。
2. 输入框聚焦时不劫持按键；快捷键提示（跟做模式底部小字或 `?` 弹出速查）。
3. 按键分发纯函数（键→动作映射，考虑输入态）配测试。i18n。

## N4：i18n 收尾 + 自查

1. 日期/数字本地化收尾：统计页与任务时间展示用 `Intl.DateTimeFormat`/`NumberFormat` 按当前语言输出（Node 内置，零依赖）。
2. 自查本批：快捷键不与浏览器/输入法冲突、分享图卡对无图/超长标题/超多食材的边界、统计对空数据的空态。
3. README（双语）功能表更新；最终 `npm test` 全绿、工作区干净、未 push、中文总结。
